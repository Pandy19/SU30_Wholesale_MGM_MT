    + Main Flow:→ Buy from Suppliers 
                → Inspect Goods 
                → Store in Stock 
                → Sell to Customers 
                → Earn Profit

    + Suppliers:→ Flow:
                → Register Supplier → Assign Brands / Products → Supplier Offers (price, qty, lead time) 
                → Create Purchase Order (PO + invoice) 
                → Goods Arrival 
                → Inspect Goods 
                → Accept (add to stock) / Reject (return, refund, or replacement)

    + Stock Management:
                → Approved Goods  Add to Stock 
                → Categorize (Category → Brand → Product) 
                → Store in Warehouse/Shelf 
                → Track Stock Levels 
                → Record Movements in Stock Ledger 
                → Monitor Low Stock

    + Customer flow:
                → Register Customer (B2B / B2C) 
                → Set Payment Rule (B2C full payment / B2B credit) 
                → Create Sales Order → Deduct Stock → Receive Payment (full or partial) 
                → Track Outstanding Balance → Generate Invoice & Sales History

    + Reports: 
                → Collect Sales, Purchase & Stock Data 
                → Generate Sales Report 
                → Calculate Cost & Profit 
                → Produce Profit Report 
                → Track Stock Movements (Stock Ledger) 
                → Review Performance & Outstanding Payments
